# Daily Routine Website

This is a simple static daily routine website built as a single `INDEX.HTML` file.

## Deploy

### GitHub Pages
1. Create a GitHub repository.
2. Add remote: `git remote add origin https://github.com/<username>/<repo>.git`
3. Push: `git branch -M main && git push -u origin main`
4. Enable GitHub Pages in repository settings, source = main branch.

### Netlify
1. Go to `https://app.netlify.com/drop`.
2. Drop the `INDEX.HTML` file.
3. Use the generated URL.

### Local preview
Open `INDEX.HTML` in your browser.
